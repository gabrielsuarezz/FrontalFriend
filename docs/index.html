FrontalFriend

A Mental Health Support Mobile Application

About the Project

FrontalFriend is a comprehensive mental health support mobile application developed during INIT Build Mobile Development Fall 2025. The app combines AI-powered chat support, relaxation features, and wellness tools to provide users with accessible emotional support and stress relief.

Built with modern mobile development technologies, FrontalFriend demonstrates the integration of artificial intelligence, media playback, real-time notifications, and cloud storage in a production-ready mobile application.

Key Features

AI Chat Support

GPT-4 powered conversational companion providing empathetic mental health support with context-aware responses.

Destress & Relax

Curated video backgrounds and calming music tracks with dynamic media playback and graceful fallback handling.

Smart Reminders

Notification system for wellness check-ins and self-care reminders with full notification permissions handling.

Physical Health Tracking

Monitor wellness metrics and maintain health records with persistent local storage.

Important Documents

Secure document storage and viewer with Supabase cloud integration for accessibility across devices.

User Authentication

Firebase authentication with login, registration, and password recovery flows.

Technology Stack

React Native Expo Expo Router OpenAI GPT-4 Firebase Supabase Expo AV (Audio/Video) Expo Notifications AsyncStorage TypeScript JavaScript

Development Highlights

Cross-Platform

Built with React Native for iOS, Android, and web deployment from a single codebase.

Modern Architecture

File-based routing with Expo Router, React hooks for state management, and modular component design.

Cloud Integration

Firebase Authentication and Supabase for backend services, enabling secure data synchronization.

User Experience

Thoughtful UI/UX with smooth animations, haptic feedback, and responsive design patterns.
